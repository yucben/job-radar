# JobRadar 🔍 — 打破求职信息差

> 你不知道的好公司，这里都能找到。资质平平也能进好公司——因为你知道别人不知道的。

## 功能

- 🏭 **多维度筛选** — 行业 / 融资轮次 / 成立时间 / 项目优势 / 所在地区 / 36氪报道 / 正在招聘
- 📋 **卡片+表格双视图** — 信息密度高，一目了然
- 🔎 **公司详情弹窗** — 融资额、投资人、团队规模、在招职位
- 💰 **薪资透明** — 每个职位标注薪资范围
- 🏆 **隐形好公司** — 重点收录有融资/有壁垒但不一定知名的高质量公司

## 为什么有用

求职最大的问题不是"找不到工作"，而是"不知道有哪些好公司"。

- 大厂人人都知道，但竞争惨烈
- B轮-C轮的隐形冠军，福利好、成长快、但鲜为人知
- 投资机构用真金白银投票的公司，往往比舆论更有参考价值

JobRadar 把融资数据、招聘信息、公司背景整合在一起，帮助你发现那些"别人不知道的好机会"。

## 数据来源

- 公开融资信息（IT桔子、36氪、投中网）
- 招聘平台公开职位
- 企业官网及媒体报道
