# 🔍 JobRadar — 打破求职信息差

> 你不知道的好公司，这里都能找到。资质平平也能进好公司——因为你知道别人不知道的。

## 🎯 解决什么问题

求职最大的信息差不是"找不到工作"，而是**不知道有哪些好公司**。

- 大厂人人都知道，竞争惨烈
- B轮-C轮的隐形冠军，福利好、成长快、但鲜为人知
- VC用真金白银投票的公司，比舆论更有参考价值

JobRadar 把融资数据、招聘信息、公司背景整合在一起，帮你发现**别人不知道的好机会**。

## ✨ 功能

- 🏭 **8维筛选** — 行业 / 融资轮次 / 成立时间 / 项目优势 / 地区 / 36氪报道 / 招聘类型 / 学历要求 + 名称搜索
- 📋 **卡片+表格双视图** — 信息密度高，一目了然
- 🔎 **公司详情弹窗** — 产品、融资额、投资人、技术栈、福利、城市分布
- 💰 **薪资透明** — 200+ 职位，每个标注薪资范围、经验、学历
- ⚠️ **避坑查询** — 脉脉/知乎/牛客/小红书一键搜负面评价
- 🎓 **校招筛选** — 应届生专属视角
- 📱 **响应式** — 手机/平板/桌面全适配

## 🏗️ 技术栈

| 层 | 技术 |
|---|------|
| 前端 | HTML5, CSS3 (CSS Variables), Vanilla JavaScript (ES6+) |
| 样式 | 自定义 Design System, CSS Grid, Flexbox, 响应式 |
| 数据 | 静态 JSON 数据结构, 客户端筛选引擎 |
| 部署 | GitHub Pages, CDN 自动加速 |
| 工具 | Git, GitHub API, Python (数据维护脚本) |

## 📁 项目结构

```
job-radar/
├── index.html          # 主页面（筛选面板 + 结果展示 + 弹窗）
├── css/
│   └── style.css       # 完整 Design System（变量/布局/组件/响应式）
├── js/
│   ├── data.js         # 40家公司 + 200+职位数据
│   └── app.js          # 筛选引擎 / 渲染器 / 交互逻辑
└── README.md
```

## 📊 数据覆盖

- **40 家公司** — C轮~上市，覆盖 20 个行业
- **200+ 职位** — 带薪资/经验/学历要求
- **数据维度** — 融资额/投资人/规模/技术栈/福利/城市/产品
- **标签体系** — 独角兽/专精特新/瞪羚企业/高新技术企业

## 🚀 本地运行

```bash
git clone https://github.com/yucben/job-radar.git
cd job-radar
python3 -m http.server 8080
# 打开 http://localhost:8080
```

纯静态，无需构建。
